# 10x Chart Style Guide

Standard visual style for all charts and dashboards generated by 10x-Analyst.

---

## Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| 10x Orange | `#FF6B35` | Primary accent, first data series, CTAs |
| Deep Blue | `#004E89` | Secondary, titles, headers |
| Emerald | `#00A878` | Positive indicators, growth, success |
| Sunshine | `#FFD166` | Warnings, third data series |
| Coral | `#EF476F` | Negative indicators, decline, alerts |
| Ocean | `#118AB2` | Additional data series |
| Midnight | `#073B4C` | Text, dark backgrounds |
| Cream | `#F5EDE0` | Dashboard background |

### Sequential Palette (for ordered data)
```python
COLORS = ['#FF6B35', '#004E89', '#00A878', '#FFD166', '#EF476F', '#118AB2', '#073B4C']
```

### Semantic Colors
```python
POSITIVE = '#00A878'  # Green — growth, profit, success
NEGATIVE = '#EF476F'  # Red — decline, loss, alert
NEUTRAL = '#004E89'   # Blue — informational
HIGHLIGHT = '#FF6B35' # Orange — attention, primary
```


## Chart Rules

### Titles
- Every chart title must include the **key takeaway**, not just the metric name
- Good: "Revenue Grew 23% MoM Driven by Product X"
- Bad: "Revenue Over Time"

### Axes
- Always label both axes
- Format numbers: use commas for thousands, `$` prefix for currency, `%` suffix for percentages
- Rotate x-axis labels 45 degrees if >6 categories

### Legends
- Position at bottom for line/bar charts
- Position at right for pie/donut charts
- Limit to 10 items max — group remainder as "Other"

### Grid
- Use light horizontal gridlines only (no vertical)
- Grid color: `#eeeeee`

### Data Labels
- Add data labels on bar charts (value at end of bar)
- Skip data labels on line charts (use tooltips in dashboard)

### Chart Sizing
| Chart Type | Figure Size | DPI |
|-----------|------------|-----|
| Line/Bar | (12, 6) | 150 |
| Heatmap | (10, 10) | 150 |
| Donut/Pie | (8, 8) | 150 |
| Scatter | (10, 8) | 150 |
| Histogram | (10, 6) | 150 |

## Dashboard CSS

```css
/* KPI Card Delta Indicators */
.delta.up { color: #00A878; }    /* Emerald for positive */
.delta.down { color: #EF476F; }  /* Coral for negative */

/* Background */
body { background: #F5EDE0; }    /* Cream */

/* Headers */
h1, h2, h3 { color: #004E89; }  /* Deep Blue */

/* Cards */
.card { background: white; border-radius: 12px; box-shadow: 0 2px 8px rgba(0,0,0,0.08); }

/* Primary accent */
.accent { color: #FF6B35; }      /* 10x Orange */
```

## Chart.js Configuration

```javascript
const COLORS = ['#FF6B35', '#004E89', '#00A878', '#FFD166', '#EF476F', '#118AB2', '#073B4C'];

// Default options for all charts
const defaultOptions = {
    responsive: true,
    maintainAspectRatio: true,
    plugins: {
        legend: { position: 'bottom', labels: { padding: 16, usePointStyle: true } },
        tooltip: { backgroundColor: '#073B4C', titleColor: '#fff', bodyColor: '#fff', padding: 12, cornerRadius: 8 }
    },
    scales: {
        x: { grid: { display: false } },
        y: { grid: { color: '#eeeeee' }, ticks: { callback: value => value.toLocaleString() } }
    }
};
```
