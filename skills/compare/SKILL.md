# 10x Analyst Loop — Dataset Comparator (Parallel Profiling)

Compare two data projects side-by-side using **parallel subagents** for profiling. Detects structural differences, metric shifts, and quality changes.

## STEP-BY-STEP INSTRUCTIONS

### STEP 0 — Smart Input Resolution for BOTH Projects

For `$0` (Project A) and `$1` (Project B):

1. If `input/ARG/` exists with data files: use it.
2. Otherwise treat ARG as a path. Extract basename. Copy files:
   ```bash
   mkdir -p input/BASENAME
   cp "ARG"/*.csv "ARG"/*.xlsx "ARG"/*.json input/BASENAME/ 2>/dev/null
   ```
3. If either project has no data files: tell user and STOP.

```
PROJECT_A = resolved name of $0
PROJECT_B = resolved name of $1
```

### STEP 1 — Profile Both Projects in Parallel (Swarm)

Create output directories:
```bash
mkdir -p output/$PROJECT_A output/$PROJECT_B
```

Spawn **2 parallel subagents** with worktree isolation to prevent file conflicts:

**SUBAGENT A — Profile Project A:**
```
Agent(
  description: "Profile $PROJECT_A",
  prompt: "Run the 10x profiler on input/$PROJECT_A:
    python ${CLAUDE_SKILL_DIR}/../../scripts/profiler.py input/$PROJECT_A output/$PROJECT_A/data-profile.md
    Read and return the full profile results.",
  isolation: "worktree",
  run_in_background: true
)
```

**SUBAGENT B — Profile Project B:**
```
Agent(
  description: "Profile $PROJECT_B",
  prompt: "Run the 10x profiler on input/$PROJECT_B:
    python ${CLAUDE_SKILL_DIR}/../../scripts/profiler.py input/$PROJECT_B output/$PROJECT_B/data-profile.md
    Read and return the full profile results.",
  isolation: "worktree",
  run_in_background: true
)
```

Wait for BOTH subagents to complete before proceeding.

### STEP 2 — Load and Compare Profiles

Read both `data-profile.json` files. For each matching file name across projects, compare:

| Metric | How to Compare |
|--------|---------------|
| Row count | Absolute diff and % change |
| Column count | Added/removed columns |
| Quality score | Point difference |
| Missing values | Per-column change |
| Duplicates | Count change |
| Numeric stats | Min/max/mean/median shifts |
| Value distributions | Top-N category changes |

### STEP 3 — Compute Metric Diffs

For files that exist in BOTH projects with similar schemas:
```python
import pandas as pd
df_a = pd.read_csv(f'input/{PROJECT_A}/file.csv')
df_b = pd.read_csv(f'input/{PROJECT_B}/file.csv')

# Row count change
row_diff = len(df_b) - len(df_a)
row_pct = (row_diff / len(df_a)) * 100

# Numeric column shifts
for col in numeric_columns_in_both:
    mean_a = df_a[col].mean()
    mean_b = df_b[col].mean()
    change_pct = ((mean_b - mean_a) / mean_a) * 100
```
