# 10x Analyst Loop — Multi-Source Research Engine

Research any topic across multiple platforms from the last 30 days. Synthesize findings into an expert briefing ranked by engagement signals.

Inspired by the last30days research methodology. Adapted and branded for 10x.in.

## STEP-BY-STEP INSTRUCTIONS

### STEP 1 — Parse User Intent

Extract from `$ARGUMENTS`:
- **TOPIC:** What they're researching (the full argument text)
- **QUERY_TYPE:** Detect from keywords:
  - "best", "top", "recommend" -> RECOMMENDATIONS
  - "news", "announced", "launched" -> NEWS
  - "prompt", "how to use", "tips" -> PROMPTING
  - Everything else -> GENERAL

Tell user:
```
Researching: {TOPIC}
Query type: {QUERY_TYPE}
Searching across: Reddit, X, YouTube, Hacker News, Web
```

### STEP 2 — Run Multi-Source WebSearch

Execute 4-6 targeted searches using WebSearch tool. Run them in PARALLEL:

**Search 1 — Reddit:**
```
"{TOPIC} site:reddit.com 2026"
```

**Search 2 — Hacker News:**
```
"{TOPIC} site:news.ycombinator.com 2026"
```

**Search 3 — YouTube:**
```
"{TOPIC} site:youtube.com 2026"
```

**Search 4 — X/Twitter:**
```
"{TOPIC} site:x.com OR site:twitter.com 2026"
```

**Search 5 — General Web (by QUERY_TYPE):**
- RECOMMENDATIONS: `"best {TOPIC} 2026"`, `"top {TOPIC} recommendations"`
- NEWS: `"{TOPIC} news 2026"`, `"{TOPIC} announcement"`
- PROMPTING: `"{TOPIC} prompts examples"`, `"{TOPIC} techniques tips"`
- GENERAL: `"{TOPIC} 2026 discussion"`, `"{TOPIC} guide"`

**Search 6 — Expert/Deep:**
```
"{TOPIC} analysis review comparison 2026"
```

### STEP 3 — Rank and Score Results

Weight sources by engagement signal strength:
1. **Reddit** (highest) — upvotes + comments prove community validation
2. **Hacker News** — points + comments prove technical validation
3. **YouTube** — views + like ratio prove broad interest
4. **X/Twitter** — likes + reposts prove real-time relevance
5. **Web articles** — publication authority, no engagement data

For each result, note:
- Platform and URL
- Title/headline
- Engagement metrics (if available)
- Key quote or finding (1-2 sentences)
- Recency (prioritize last 30 days)

### STEP 4 — Detect Cross-Platform Convergence

Look for the SAME topic, product, or opinion appearing across multiple platforms. These are the strongest signals:
- Same product recommended on Reddit AND Hacker News = very strong
- Same news on X AND web articles = confirmed trend
- Same technique on YouTube AND Reddit = validated approach

### STEP 5 — Synthesize Briefing

Write `output/research/TOPIC-slug.md` (create `output/research/` if needed):

### STEP 6 — Present to User

Show the key findings summary inline (not just the file path). Then:
```

Want me to:
- Dive deeper into any finding?
- Compare specific products/tools mentioned?
- Write a prompt based on the best techniques found?
```
