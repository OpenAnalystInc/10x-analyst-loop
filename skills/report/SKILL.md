# 10x Analyst Loop — Report Generator

Generate a publication-ready Markdown analysis report from any data project.

## STEP-BY-STEP INSTRUCTIONS

### STEP 0 — Smart Input Resolution

1. If `input/$0/` exists with data files: use it. PROJECT = `$0`.
2. Otherwise treat `$0` as a path. Extract folder name as PROJECT. Copy files:
   ```bash
   mkdir -p input/PROJECT
   cp "$0"/*.csv "$0"/*.xlsx "$0"/*.xls "$0"/*.json input/PROJECT/ 2>/dev/null
   ```
3. If no data files anywhere: tell user and STOP.

### STEP 1 — Data Engineering
```bash
mkdir -p output/$PROJECT/charts output/$PROJECT/cleaned-data
python ${CLAUDE_SKILL_DIR}/../../scripts/profiler.py input/$PROJECT output/$PROJECT/data-profile.md
python ${CLAUDE_SKILL_DIR}/../../scripts/data_cleaner.py input/$PROJECT output/$PROJECT/cleaned-data
```

### STEP 2 — Statistical Analysis
Load cleaned data. Compute KPIs, trends, segments, correlations. Save to `output/$PROJECT/insights.json`.

### STEP 3 — Generate Charts
For each insight, create a supporting chart. Save PNGs to `output/$PROJECT/charts/`.

### STEP 4 — Write Report
Write `output/$PROJECT/report.md`:
```markdown
# Data Analysis Report
> Generated by **10x Analyst Loop** | {date} | Source: input/{project} | Session: ${CLAUDE_SESSION_ID} | Powered by 10x.in

## Executive Summary
- {Finding 1: metric + direction + magnitude}

## Data Overview
| Metric | Value |
|--------|-------|

## Key Findings
### 1. {Headline}
{Numbers + context}
![Chart](charts/chart_name.png)
**Implication:** {Business meaning}

## Recommendations
1. **{Action}** -- Finding #X -> Expected impact

## Methodology
## Appendix
---
*Generated by [10x Analyst Loop](https://10x.in) v2.0.0*
```

### STEP 5 — Strategy Layer
Prioritize P0-P3. Refine recommendations. Add executive brief.

### STEP 6 — Distribute Report (if MCP configured)

IF `output/$PROJECT/.mcp-config.json` exists with messaging targets:
- **Gmail MCP**: Use ToolSearch to find Gmail tool, email report.md as attachment
- **Slack MCP**: Use ToolSearch to find Slack tool, post summary with link to full report
- **Discord MCP**: Use ToolSearch to find Discord tool, post summary
IF no config: skip silently.

### STEP 7 — Present to User
Show report location, top insights, suggest `:dashboard $PROJECT`.

## Examples
```
/10x-analyst-loop:report my-sales
/10x-analyst-loop:report ~/exports/q1 "focus on customer retention"
```
